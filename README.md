Student Portal - A web-based student portal system built with PHP and MySQL. This platform provides students with access to academic resources and profile management.
Features:
- User authentication system for students and administrators
- Student profile management
- Course information and materials access
- Database integration for student records
- Responsive web interface

Technologies Used:
- PHP
- MySQL
- HTML/CSS
- JavaScript

Setup:
1. Clone the repository
2. Import database_setup.sql to your MySQL server
3. Configure database connection in config.php
4. Deploy to a web server with PHP and MySQL support
